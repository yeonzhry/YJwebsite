Start Page

I placed a custom-made logo on the start page, with a glitch effect applied using Post-processing effects in Three.js. When the 'click to start' button is clicked, it navigates to a screen with other page.

Photo & Viedo Grid

This is a CCTV-themed grid made using photos and videos I took. When you hover over each grid, an enlarged version of the image appears.

Audio Visualizer

This is an audio visualizer that visualizes sound. When the 'click to start' button is pressed, the audio starts playing, and the visualizer activates. You can also visualize your own voice by pressing the microphone button, or upload files from your computer to use. Additionally, you can customize the visualizer's color, blur, and transparency using the debugger.

3D Modeling

This is a 3D model I created using Blender. I added it to the page using Three.js and also included animations.

Synthesizer

This is a synthesizer made using Tone.js. You can play it by clicking each key with the mouse or by pressing the corresponding keys on the keyboard. You can also adjust the volume, and select the type of synthesizer and effects.

Video in 3D TV Model

I added a video produced in the 'Creativity & Visual Expression' class to a 3D TV model. I used Three.js to add both the model and the video to the page, and applied a glitch effect using Post-processing effects in Three.js.

About me & Code File

This is a page where I added an image introducing myself. There are 'code file' and 'Instagram' buttons that link to their respective pages, and clicking the image replicates it on the screen.
